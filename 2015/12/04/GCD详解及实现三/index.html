iOS绘图功能

iOS 系统层级介绍 iOS大致分为四层：Core OS层、Core Services层、Media层以及Cocoa Touch层。

Core OS

最底层功能提供了文件系统、线程管理、网络、I/O、内存管理、数学计算、安全服务、外部设备管理等。

Core Service层

提供了大量系统服务，包括GCD、应用内购买、位置服务、Sqlite、xml支持、字符串管理,如同 Core OS它也扮演幕后角色

Media层

提供了图像、声音、视频技术，将多媒体体验带到移动平台上。

Cocoa Touch层

包括了多任务、数据保护、推送通知、本地通知、设备间通讯、手势识别、标准视图控制器，应用与用户交互大多发生在这一层。

iOS Media层图形技术

iOS中图形技术分两种：OpenGL ES、核心图库（Core Graphic）,OpenGL是苹果支持的，开发2D或3D程序的一些列跨平台API，也是业界标准，广泛应用于游戏开发。
另外Core Graphic是iOS原生的核心图库，与Open GL相比，更适合绘制简单的几何图形，也更适合于面向对象开发，非常方便在UIView上通过drawRect：方法完成绘图。

Quartz 2D

Quartz 2D是基于向量的二维绘图引擎。基于向量是指使用Quartz 2D绘图时，往往使用Quartz 2D的坐标系（用户空间）而非视图的坐标系。Quartz2D坐标原点与iOS系统坐标系是不同的，
一个在左上角一个在左下角。
CTM当前变化举证，将绘图用户空间映射到设备空间。

贝济埃路径

UIBezierPath 采用系统坐标系，因此没有任何坐标系转换的麻烦，另外它是在Core Graphic基础上封装而成，因此拥有Core Graphic主要功能，却无需考虑图形环境。
